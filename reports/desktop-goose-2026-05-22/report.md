## 1. 总定位

Desktop Goose 是一款由独立开发者 Samperson（Sam Chiet）于 2020 年 1 月在 itch.io 平台发布的桌面宠物软件。产品定位是"一只会捣乱的桌面鹅"——它并非传统意义上的生产力工具，而是一款以娱乐、解压和恶作剧为核心目的的桌面玩具。

产品采用"Pay What You Want"定价模式，用户可免费下载，也可自愿付费支持开发者。支持 Windows 和 macOS 双平台。

核心交互逻辑非常简单：安装后，一只像素风格的白色大鹅会在用户的屏幕上游走，执行一系列预设的"捣蛋"行为，包括：
- 在屏幕上随机走动，留下泥脚印
- 拖拽用户打开的图片、窗口和文本
- 从屏幕边缘"叼"来备忘录/便签纸丢在桌面上
- 发出鹅叫声
- 用喙"啄"用户的鼠标光标

产品走红的主要原因是其反差感——在严肃的桌面工作环境中，一只捣乱的鹅带来了意想不到的幽默效果。2020 年初在社交媒体（Twitter、Reddit、Bilibili 等）上引发了大量 UGC 传播。

## 3. 各界面功能与评价

### 3.1 桌面主界面（无窗口模式）

Desktop Goose 最显著的设计特点是**没有传统应用主窗口**。鹅直接在桌面上活动，跨越所有应用窗口的上方（置顶显示）。这种设计选择强化了"鹅是桌面环境的一部分"的沉浸感。

鹅的动画采用像素风格，动作包括：行走、奔跑、 idle 待机、叼东西、拖拽物品。行走时会留下灰色的泥脚印，一段时间后自动消失。

**评价**: 无窗口设计是产品的核心创新点，但也带来了一些交互上的限制——用户无法像常规应用一样"最小化"或"关闭"鹅，只能通过系统托盘或任务管理器退出。

### 3.2 设置面板

通过系统托盘右键菜单可打开设置面板， reportedly 包含以下配置项：
- 鹅的出现频率/活跃度
- 是否启用声音
- 是否启用脚印
- 是否启用拖拽行为
- 自定义备忘录内容（支持读取文本文件）
- 备忘录出现频率

**评价**: 设置选项相对精简，符合产品的"轻量玩具"定位。开发者 reportedly 在后续更新中加入了 Mod 支持，允许用户自定义鹅的外观和行为。

### 3.3 备忘录/便签系统

鹅会从屏幕边缘叼来白色便签纸丢在桌面上。便签内容可通过配置文件自定义， reportedly 支持从指定文本文件中随机读取句子。

**评价**: 便签系统是鹅与用户"交流"的主要方式，增加了互动趣味性。但便签不支持用户直接编辑或交互（只能被鹅叼走或关闭）。

### 3.4 系统托盘

Windows 版在系统托盘显示鹅图标，右键菜单提供：Settings（设置）、Close（退出）等选项。

**评价**: 托盘图标是用户控制鹅的主要入口。由于鹅本身没有窗口标题栏，托盘菜单是唯一的"常规"退出路径。

## 4. UI/UX 评价

### 4.1 视觉风格

产品采用复古像素艺术风格，鹅的设计简洁但辨识度高——白色身体、橙色喙、黑色眼睛。脚印、便签等视觉元素风格统一，整体呈现出一种" indie 游戏"的美学质感。

### 4.2 交互设计

- **无窗口模式**: 创新的沉浸式设计，鹅仿佛真的活在桌面上。但缺乏常规窗口控件，新用户可能需要时间适应。
- **不可预测性**: 鹅的行为带有随机性，增强了"宠物感"和惊喜感，但也可能在用户专注工作时造成干扰。
- **声音反馈**: 鹅叫声 reportedly 采用真实的鹅叫录音，增加了真实感和喜剧效果。

### 4.3 用户体验问题

- **干扰性**: 鹅会拖拽窗口和文件，在需要精确操作的工作场景中可能造成困扰。
- **退出路径不明显**: 没有窗口标题栏的关闭按钮，部分用户 reportedly 初次使用时找不到退出方法。
- **平台差异**: macOS 版 reportedly 功能相对 Windows 版有所简化。

### 4.4 可访问性

产品未提供无障碍相关功能（屏幕阅读器支持、高对比度模式等），考虑到其"桌面玩具"的定位，这在预期之内。
