## 1. 详细的产品逻辑介绍

### 1.1 总定位

AnyRouter 是一个面向中国开发者的第三方 Claude Code / Claude API 代理中转服务。用户通过修改本地环境变量,将 Claude Code 的 API 请求指向 AnyRouter 的端点(如 `https://c.cspok.cn`),从而在不使用国际网络代理的情况下访问 Anthropic 的 Claude 模型。产品基于开源项目 NewAPI 搭建,定位为公益性质的 AI API 聚合中转站。

### 1.2 界面清单

由于官网不可访问且产品本身为纯 API 代理服务(无桌面应用、无移动端 App),实际可采集的界面有限:

| 2 | 第三方介绍页 | https://www.rcdfrd.com/archives/40 | 使用教程、风险提示 | [02_web_blog_detail.png](screenshots/02_web_blog_detail.png) |

### 1.3 各界面功能与评价

#### 1.3.1 官网首页(不可访问)

- **功能**:根据第三方信息,官网应提供用户注册、API 密钥生成、额度查询、签到领取免费额度等功能
- **交互**:用户访问官网 → 注册账号(早期支持 GitHub 登录,后期改为 LinuxDO 论坛账号或 `.edu.cn` 教育邮箱) → 获取 API 密钥 → 配置本地环境变量
- **评价**:2026-05-22 官网完全不可访问,HTTPS 连接返回 `SSL_ERROR_NO_CIPHER_OVERLAP`,HTTP 返回 Cloudflare DNS 错误 1001。这表明域名可能已停用或 DNS 配置出现问题。对于一个依赖线上服务的 API 中转站,官网不可访问是严重的基础设施问题
- **截图**:[01_web_homepage.png](screenshots/01_web_homepage.png)

#### 1.3.2 第三方介绍页(rcdfrd blog)

- **功能**:详细介绍 AnyRouter 的使用方法,包括三个必需的环境变量配置、注册流程、风险提示

- **评价**:信息组织清晰,以 Markdown 格式呈现,包含代码块、表格和列表。文章明确指出了这是第三方服务,并附带了风险提示。但博客非官方渠道,信息可能存在滞后(发布于 2025-12-07)

## 2. UI/UX 风格和质量描述

### 2.1 视觉风格

由于官网不可访问,无法直接评估官方网站的视觉风格。从第三方博客页面的截图来看,AnyRouter 的使用方式完全基于命令行环境变量配置,无图形用户界面。用户在终端中设置三个环境变量后即可使用 Claude Code,整个过程无视觉交互。

### 2.2 信息密度与层级

官网无法访问,无法评估。从使用流程来看,产品将复杂的中转逻辑简化为三个环境变量,对开发者友好。但用户需自行寻找教程(如第三方博客)来了解配置方法,官方文档不可达。

### 2.3 交互流畅度

产品无 GUI,交互完全通过命令行完成。配置流程简洁:设置环境变量 → 运行 `claude` 命令。但由于官网不可访问,新用户目前无法注册或获取 API 密钥,产品实际上处于不可用状态。

### 2.4 文案质量

第三方博客中的文案为中文,表述清晰。环境变量说明使用表格呈现,易于理解。风险提示部分明确指出了这是第三方服务,建议用户谨慎传输敏感代码和密钥。

### 2.5 可访问性观察

CLI 工具的可访问性取决于终端本身。对于习惯命令行的开发者而言无障碍,但对于非技术用户门槛较高。

## 4. 定价

AnyRouter 的定价模式较为特殊:

- **计费方式**:公益性质,不支持充值,完全依赖免费额度
- **免费额度获取**:
  - 普通注册:赠送 $50
  - 邀请链接注册:可达 $100
  - 每日签到:$10-$25/天(不同时期额度不同)
  - 邀请好友:双方各得 $50
- **模型定价**:早期有资料显示 Claude Sonnet 约 $3/$15 per 1M tokens(与官方原价持平),但此为消耗额度的速率,非实际付费

> 注意:由于官网不可访问,以上定价信息均来自网络搜索和第三方博客,时效性无法保证。

## 5. 目标用户

基于第三方信息推断:

- **主要用户**:中国境内的开发者,尤其是无法直接访问 Anthropic 官方 API 的用户
- **使用场景**:个人学习、轻度开发、体验 Claude Code 功能
- **技术门槛**:需要具备命令行操作能力,能配置环境变量
- **不适合**:企业级生产环境(缺乏 SLA 保障)、对数据隐私要求极高的场景(请求经过第三方服务器)
